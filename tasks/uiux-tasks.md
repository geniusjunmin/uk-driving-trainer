### [UI-002] 设计 HUD 仪表盘与转向指示器的动态视觉样式
* **任务状态**: DONE
* **优先级**: HIGH
* **依赖关系**: [UI-001]
* **输入**: `需求.txt` 中“速度表用 mph”及“转向灯”需求。
* **输出**:
  * 创建/更新 `src/ui/hud.css`。
* **验收标准**:
  * 速度表以大字号数字显示，辅以半圆弧线模拟传统指针。
  * 转向指示灯（左 Q/右 E 箭头）在激活状态下有高亮闪烁动画（采用 CSS keyframes @keyframes blink 呼吸闪烁）。
  * 整体布局不遮挡驾驶视线，通常位于屏幕底部或右上角。
* **执行步骤**:
  1. 规划 HUD 的 CSS 网格 (Grid) 与弹性盒 (Flexbox) 布局。
  2. 编写闪烁动画 `@keyframes`，设置呼吸频率。
  3. 采用 `backdrop-filter: blur(10px)` 实现拟物毛玻璃效果。
* **完成后需要修改的文件**:
  * `src/ui/hud.css`
* **完成后需要记录的日志**:
  * "UI-002: 完成了仪表盘与转向指示器的 CSS 动画与毛玻璃布局样式设计。"
